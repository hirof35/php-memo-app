# 多機能PHPメモ帳 (PHP & SQLite Notepad)

PHPとSQLiteを使用した、軽量でシンプルなWebメモ帳アプリケーションです。
データベースの面倒な初期設定が不要で、ファイルをサーバーに配置するだけで一元管理されたメモ帳がすぐに動作します。

## 🚀 主な機能

- **メモの残す（Create）**: テキストエリアから手軽にメモを保存。改行もそのまま保持されます。
- **メモを見る（Read）**: 保存されたメモを最新順に一覧表示します。
- **メモを直す（Update）**: 過去に書いたメモの内容をその場で編集・上書き更新できます。
- **メモを消す（Delete）**: 不要になったメモを個別にピンポイント削除（確認ダイアログ付き）。
- **メモを探す（Search）**: キーワードを入力して、過去のメモから該当するものを瞬時にあいまい検索（絞り込み）できます。

## 🛠️ 技術スタック / 仕様

- **言語**: PHP 7.4以上（PHP 8.x 推奨）
- **データベース**: SQLite3 (PDO経由で接続)
- **セキュリティ対策**: 
  - XSS（クロスサイトスクリプティング）対策：`htmlspecialchars`によるサニタイズ
  - SQLインジェクション対策：プリペアドステートメント（プレースホルダー）の徹底
- **デザイン**: 外部CSSフレームワーク不使用のピュアCSS（レスポンシブ対応）

## 📦 構成ファイル

本アプリは1ファイル構造（シングルページアプリケーション構成）になっています。

```text
📁 プロジェクトフォルダ/
└── 📄 index.php      # メインプログラム（ロジック・画面UI）
※アプリを起動すると、同じフォルダ内にデータベースファイル memo.db が自動生成されます。

💻 導入・動かし方
1. ファイルの準備
ローカル開発環境（XAMPP、MAMP、Dockerなど）またはWebサーバーの公開ディレクトリ（htdocs や public_html）にフォルダを作成し、その中に index.php を配置します。

2. アクセス
ブラウザを開き、配置した環境のURL（例: http://localhost/あなたのフォルダ名/index.php）にアクセスします。

3. 注意点（パーミッション）
Linuxサーバー等にデプロイする場合、アプリケーションがデータベースファイルを自動生成・書き込みできるように、フォルダの書き込み権限（パーミッション：755 または 777など）を与えてください。
